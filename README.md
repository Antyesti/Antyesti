## About

I started this project to build a **data-driven personal Pokédex** for tracking my own Pokémon collection, but I thought it would be fun to share it with everyone.

The Pokédex is designed to run entirely **locally** on your device. There's no installation, no local server, and no complicated setup required. Simply open it in a modern web browser and you're ready to go. Most features also work completely offline, although a few optional features require an internet connection. Those should be fairly self-explanatory when you use them.
